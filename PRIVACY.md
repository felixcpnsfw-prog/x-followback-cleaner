# Privacy Policy

Last updated: May 15, 2026

X Followback Cleaner is a local-first Chrome extension and CLI for comparing X/Twitter followers and following lists.

## Data the Extension Processes

The extension may process:

- X/Twitter usernames and display names visible on followers/following pages
- Snapshot metadata such as capture time, source URL, and account counts
- User-provided protected handles
- Unfollow batch progress and per-account result status

## Local Storage

The extension stores snapshots, settings, and batch progress in `chrome.storage.local` on your device. This data is not sent to a server operated by the extension developer.

## X Session Usage

Fast API unfollow mode uses your logged-in X web session from the active X tab to request unfollow actions. The extension does not ask for your X password and does not collect your X credentials.

## Downloads

When you export CSV results, the extension creates a local file through Chrome's downloads API.

## Donations

The popup contains optional creator support links, including Buy Me a Coffee and a stablecoin address. Opening those links or sending funds is optional and handled outside the extension.

## Remote Code

The extension does not load or execute remotely hosted code.
