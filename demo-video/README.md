# Demo Video

This HyperFrames composition renders a 62-second operation walkthrough for X Followback Cleaner.

It covers:

- Manual installation from GitHub with `chrome://extensions`
- `Open + Scan` followers/following snapshot flow
- Zoom-assisted scanning and completion hints
- Compare, protected handles, CSV export, and Fast API unfollow
- On-page pause/stop overlay and local-first privacy posture
- Creator attribution for [@ProfitKatze](https://x.com/ProfitKatze)

## Render

```bash
cd demo-video
npm run render:demo
```

The rendered MP4 is written to:

```text
../dist/x-followback-cleaner-demo.mp4
```

## Validate

```bash
cd demo-video
npm run check
```
