X Followback Cleaner

不用依賴 Unfollr，也不用上架 Chrome 商店。手動載入 extension，就能掃描、比對、匯出與批次退追。

Scan followers Compare snapshots Export CSV Pauseable unfollow
Step 1

先從 GitHub 載入插件

不用付 Chrome Web Store 上架費。下載 repo 後，到 Chrome extensions 開啟 Developer mode，選擇 Load unpacked。

chrome://extensions
01
Developer mode 右上角打開開發者模式。
02
Load unpacked 選擇 x-followback-cleaner/extension。
03
Pin extension 固定在工具列，之後在 x.com 直接使用。
